Introduction
This web site is intended help you learn to estimate the shape of a molecule once you have determined the molecule's Lewis Structure.  The model used is often called the VSEPR (Valence Shell Electron Pair Repulsion) model and descriptions of it may be found in almost all introductory chemistry texts.

Software Requirements
This site should work in any modern browser with javascript activated (the default). You may also download the 3-D molecular images (*.pdb files) and use the free program Jmol (Jmol requires Java to be installed on your machine) to view them.
